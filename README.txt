# BMI Calculator

A simple graphical BMI (Body Mass Index) calculator built using Python and `tkinter`. This tool allows users to calculate their BMI based on weight and height and provides a category classification (Underweight, Normal weight, Overweight, or Obesity). 

## Features
- Input for weight (in kilograms) and height (in centimeters).
- Calculates BMI using the standard formula.
- Displays BMI category.
- Error handling for invalid inputs.
- Stores and displays the last five BMI calculations for user reference.
